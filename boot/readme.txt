引导过程和引导文件布局参考linux-0.11完成。Image文件中引导文件boot.s和setup.s布局：+-----------+  	<----- 0x0|			|| boot.s    |     	1个扇区大小，512字节|			|+-----------+  	<----- 0x200 |			|| setup.s	|     	2个扇区大小，1024字节|			|+-----------+	<----- 0x600|     		||	kernel	|		最大不超过64k字节|			|+-----------+   <----- 0x10600注：其中setup.s 还记录了硬盘chs参数和当前内存大小。setup.s和boot.s在内存中的布局：+-----------+   <----- 0x0|			||	... 	|+-----------+   <----- 0x90000|			||	boot.s	||			|+-----------+	<---- 0x90200|			||	setup.s	|	|			|+-----------+	<---- 0x90600|			||	...		|+-----------+
